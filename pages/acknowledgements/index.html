Acknowledgements

The Gaius platform would not be possible without some incredible open-source projects. The Gaius engine leverages the following libraries:

Markdig is an incredible open-source markdown processing engine for .NET. It also provides numerous extensions that can enhance the capabilities of the source markdown data.

Fulid is an open-source .NET template engine that is as close as possible to the Liquid template language.

YAML DotNet

YamlDotNet is an excellent .NET based library for processing YAML.
